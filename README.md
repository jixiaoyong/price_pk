# 性价比大PK (Price PK)

一个极其轻量、秒开的单位价格对比工具。帮助您在面对不同规格、不同价格的商品时，快速选出最划算的一款。

此项目已从 Flutter Web 迁移至 Vue 3 + Vite，体积减小了 95% 以上，并支持 PWA 离线使用。

## ✨ 核心功能

- **多模式比对**：支持重量（mg, g, kg, t）和体积（ml, l）两种模式。
- **智能高亮**：实时计算单位价格，并自动高亮显示最划算的商品。
- **平滑动画**：支持一键排序，卡片位移伴随平滑过渡动画。
- **一键分享**：
  - **链接分享**：数据通过 LZString 压缩编码进 URL，复制链接即可分享当前所有对比数据。
  - **截图分享**：内置截图功能，可将对比结果生成图片保存。
- **PWA 支持**：可安装至手机桌面，支持离线使用，无缝更新特性。

## 🛠️ 技术栈

- **框架**: Vue 3 (Composition API)
- **构建**: Vite
- **状态管理**: Pinia
- **离线增强**: Vite PWA Plugin
- **工具库**: LZ-String (数据压缩), Html2Canvas (截图), Lucide-vue-next (图标)

## 🚀 快速开始

### 开发环境

1. **安装依赖**:

   ```bash
   npm install
   ```

2. **启动开发服务器**:
   ```bash
   npm run dev
   ```

### 生产构建

1. **构建项目**:

   ```bash
   npm run build
   ```

   构建产物将存放在 `dist` 目录。

2. **本地预览构建产物**:
   ```bash
   npm run preview
   ```
